Welcome to Piecemaker2

What is it?

Piecemaker is a multi-user application which assists in notating and scoring dance, theatrical pieces and almost anything time based. Its capture mode helps you keep track of what happened in rehearsals.

Features

  • Annotate On Time
  • Network Allows Simultaneous User Input
  • Network Access For External Software

Who uses it?

Piecemaker1 was commisioned by William Forsythe for use in The Forsythe Company and developed by David Kern.

PM2GO is a simple to use annotation prototype that was developed at H_Da during Motion Bank phase 1 (2010-2013). It is being used in education and by independant praticioners.

Piecemaker2 application, the one you are looking at, is an unreleased in-development prototype of a more powerful system that is meant to become Piecemaker3 one day. This version is currently only being used internally by Motion Bank.
